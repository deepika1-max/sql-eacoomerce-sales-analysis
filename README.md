# 📊 E-Commerce Sales Analysis (SQL Project)

## 📌 Project Overview

This is a beginner-level SQL Data Science project where I designed and analyzed an e-commerce database.

## 🛠 Tools Used

* SQL Server
* Structured Query Language (SQL)

## 📂 Database Tables

* customers
* products
* orders
* order_details

## 📊 Analysis Performed

* Total Revenue
* Monthly Sales Trend
* Top Customers
* Best-Selling Products

## 🎯 Key Insights

* Electronics generated highest revenue
* Delhi customers contributed most sales
* Repeat customers spent more
